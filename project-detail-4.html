Chococat.com Web Design

Chococat.com 網站設計

Category

UI/UX Design

Date

2026 / 02

A final project for the web design course in the second semester of sophomore year, I created an e-commerce website specializing in cat pet supplies, named Chococat.com. The goal of this project was to design an attractive website that provides a user-friendly shopping experience and showcases various cute cat products. Through this project, I learned how to use HTML and CSS to create a visually appealing and functional e-commerce platform.

大二下學期網頁設計的期末專案，我做的主題為專門販售貓咪寵物用品的電商網站，名為Chococat.com。這個專案的目標是設計一個吸引人的網站，提供使用者友好的購物體驗，並且展示各種可愛的貓咪產品。透過這個專案，我學習了如何利用HTML和CSS，設計出一個既美觀又實用的電商平台。
